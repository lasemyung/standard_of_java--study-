# 접근 제어자 ( access modifier)

- private : 같은 클래스 내에서만 접근이 가능하다

- (default) 같은 패키지 내에서만 접근이 가능하다

- protected : 같은 패키지 내에서 그리고 다른 패키지의 자손 클래스에서 접근이 가능

- public : 접근 제한이 전혀 없다

  |  제어자   | 같은 클래스 | 같은 패키지 | 자손 클래스 | 전체 |
  | :-------: | :---------: | :---------: | :---------: | :--: |
  |  public   |      o      |      o      |      o      |  o   |
  | protected |      o      |      o      |      o      |      |
  |  defualt  |      o      |      o      |             |      |
  |  private  |      o      |             |             |      |
